- creation-acceptance-gate-hook.py：無 DOC 類型豁免邏輯（不需修改）
- Phase 4 的 DOC 類型豁免（TDD 流程）：與建立後審核無關，不需修改

移除 2 個檔案中建立後審核的 DOC 類型豁免條件：

1. ticket-lifecycle.md（v5.4.0 → v5.5.0）：
   - 從豁免條件表格中移除「DOC 類型（任務範圍單純）| 純文件更新，可跳過」行
   - 僅保留「已審核父 Ticket 的子任務」豁免
