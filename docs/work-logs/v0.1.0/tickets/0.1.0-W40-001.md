## Problem Analysis

用戶回報三個框架初始化引導缺口：

**缺口 1：docs 結構未強制建立**
- 現有 project-init Skill 有要求建立 docs/ 及子目錄，但未強制執行
- 新使用者下載框架後可能缺少 docs/work-logs/ 等目錄

**缺口 2：.gitignore 未設定**
- 下載框架後未自動設定 .gitignore
- 導致 .claude/hook-logs/、.coverage 等大量日誌檔列入 git 變更

**缺口 3：Hook 阻擋後操作說明不足**
- skip-gate Hook 阻擋操作時（如在 docs/ 新增文件），錯誤訊息只說「不允許」
- 未提供正確的替代操作路徑（如：應建立 Ticket、使用對應代理人）
- AI 被 Hook 阻擋後找不到正確做法，造成卡住
