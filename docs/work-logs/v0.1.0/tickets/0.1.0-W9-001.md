# Execution Log

## Task Summary

修復 _execute_list() 遇到 HandoffDirectionUnknownError 中斷整個 --list 的設計缺陷

---

## Problem Analysis

<!-- To be filled by executing agent -->

---

list_pending_handoffs() 在遇到不合法 direction 時執行 raise HandoffDirectionUnknownError，導致 for 迴圈完全中止，後續有效 handoff 全部遺失。_execute_list() 雖然 catch 了 exception，但此時函式已中斷，只能回傳空結果。
## Solution

<!-- To be filled by executing agent -->

---

將 list_pending_handoffs() 中的 raise 改為 per-file 處理：schema_error_count += 1 + print warning to stderr + continue。更新 _execute_list() 的 except 區塊為安全網說明，移除 docstring 中的 Raises 記錄。

<!-- To be filled by executing agent -->

---

737/737 測試通過（含更新的 test_unknown_direction_skipped_with_warning）。行為驗證：N 個有效 + 1 個損壞 handoff → 顯示 N 個有效，損壞的輸出 stderr 警告。
