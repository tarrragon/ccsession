## Problem Analysis

根據 parallel-evaluation A Quality 視角檢查，hook_messages.py 中 ValidationMessages 類別包含以下硬編碼 emoji，違反 language-constraints.md 規則 3：

已確認的位置：
- 第 435 行：MCP_TESTS_ERROR_TITLE = "❌ MCP run_tests 使用規範違規"

- 第 449 行：MCP_TESTS_RECOMMENDED_HEADER = "📋 推薦方案:"

## Solution

| ✅ | [OK] | MCP_TESTS_CORRECT_HEADER |
| 📋 | [推薦方案] | MCP_TESTS_RECOMMENDED_HEADER |
| 📚 | [相關規範] | MCP_TESTS_REFERENCE_HEADER |

1. 建立特性分支 `feat/w23-007-emoji-cleanup`
