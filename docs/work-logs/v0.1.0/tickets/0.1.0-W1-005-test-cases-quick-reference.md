## 邊界條件補充清單

| 邊界條件 | 描述 | 預期行為 |
|---------|------|---------|
| **空依賴** | --remove 所有依賴後 | blockedBy 為空清單 |
| **自引用** | A blockedBy A | 循環依賴錯誤，拒絕 |
| **雙向依賴** | A blockedBy B + B blockedBy A | 檢測循環，警告/拒絕 |
| **版本不匹配** | 引用其他版本 Ticket | 警告或跨版本支援 |
| **多層嵌套** | A blockedBy B blockedBy C | 逐層驗證 |
| **特殊字元** | Ticket ID 含空格/特殊字元 | 驗證和拒絕 |
