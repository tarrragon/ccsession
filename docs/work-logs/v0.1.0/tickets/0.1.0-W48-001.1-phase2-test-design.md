## 測試目標

針對 `layer-boundary-validator-hook.py` 設計完整的單元測試和集成測試套件，驗證 Hook 能準確檢測 7 大禁止項，同時排除合法上下文中的違規內容。

**測試覆蓋目標**（基於 Phase 1 驗收條件）：

| 驗收條件 | 測試覆蓋 | 檢查方式 |
|---------|--------|---------|
| **V1** | 7 大禁止項正則表達式可驗證 | 每個禁止項至少 3 個測試案例通過 |
| **V2** | 排除規則正確實現 | 在排除情境中，違規內容應被跳過 |
| **V3** | 現有 23 個 Layer 1 檔案掃描無誤報 | 集成測試在實際檔案上驗證誤報率 < 5% |
| **V4** | Hook 警告訊息格式清晰 | 測試輸出格式包含行號、禁止項類型、替代用語 |
| **V5** | Hook 正確識別 Layer 1 檔案 | 測試檔案路徑識別邏輯，確保不掃描 Layer 2 |
