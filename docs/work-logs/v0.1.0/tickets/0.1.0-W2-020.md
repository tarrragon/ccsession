# Execution Log

## Task Summary

重構 _print_recommendation() 過長函式拆分與三分支重複消除

---

## Problem Analysis

原始 `_print_recommendation()` 函式存在兩個主要問題：

1. **過長函式**：原始函式 54 行，認知負擔指數 14（遠超過閾值 10）
2. **結構重複**：to-child / to-sibling / to-parent 三個分支結構完全相同（15 行重複），只在訊息常數和參數名稱上有差異
3. **命令列印複雜化**：命令列印區塊使用直接字串串接（`+` 運算符）來拼接執行命令和註解

核心設計缺陷：如果未來新增 handoff 方向，需重複相同的 5 行程式碼，違反 DRY 原則。

---

## Solution

**整體策略**：使用三層重構方案消除所有重複

### 1. 模組級映射表 `_DIRECTION_MESSAGE_MAP` (新增於 handoff.py:60-65)

```python
_DIRECTION_MESSAGE_MAP = {
    "to-child": (HandoffMessages.RECOMMENDATION_ENTER_CHILD, "child_id"),
    "to-sibling": (HandoffMessages.RECOMMENDATION_SWITCH_SIBLING, "sibling_id"),
    "to-parent": (HandoffMessages.RECOMMENDATION_RETURN_PARENT, "parent_id"),
}
```

**優點**：
- 消除三分支 if/elif 的結構重複（15 行 → 3 行）
- 方向到訊息模板的映射清晰可見
- 新增方向只需修改映射表，無需修改函式邏輯

### 2. 共用子函式 `_print_direction_recommendation()` (新增於 handoff.py:703-713)

提取三個分支的共同邏輯：

```python
def _print_direction_recommendation(rec: Recommendation, msg_template: str, param_name: str) -> None:
    """列印方向建議（to-child/to-sibling/to-parent）。使用映射表消除重複。"""
    kwargs = {param_name: rec.next_target_id}
    print(format_msg(msg_template, **kwargs))
    if rec.next_target_title:
        print(format_msg(HandoffMessages.RECOMMENDATION_TITLE, title=rec.next_target_title))
    print(format_msg(HandoffMessages.RECOMMENDATION_REASON, reason=rec.reason))
```

**優點**：
- 統一參數名稱處理（使用 kwargs 動態構建）
- 邏輯清晰，認知負擔低

### 3. 命令列印輔助函式 `_print_execute_command()` (新增於 handoff.py:716-737)

提取命令列印邏輯，使用 format_msg() 消除字串串接：

```python
def _print_execute_command(rec: Recommendation) -> None:
    """列印建議執行的命令，根據方向添加對應註解。"""
    if not rec.command:
        return

    # 選擇註解文字
    if rec.direction == "wait" and rec.blocked_by:
        comment = HandoffMessages.RECOMMENDATION_EXECUTE_COMMENT_CHECK
    elif rec.direction == "completed":
        comment = HandoffMessages.RECOMMENDATION_EXECUTE_COMMENT_COMPLETE
    else:
        comment = None

    # 列印命令
    if comment:
        print(format_msg(HandoffMessages.RECOMMENDATION_EXECUTE_WITH_COMMENT, command=rec.command, comment=comment))
    else:
        print(format_msg(HandoffMessages.RECOMMENDATION_EXECUTE, command=rec.command))
```

**優點**：
- 消除 `+` 字串串接，統一使用 format_msg()
- 註解文字選擇邏輯清晰（Guard Clause）
- 添加新訊息模板 `RECOMMENDATION_EXECUTE_WITH_COMMENT` 於 command_lifecycle_messages.py:88

### 4. 簡化主函式 `_print_recommendation()` (修改於 handoff.py:740-777)

- 刪除內部映射表定義，改用模組級 `_DIRECTION_MESSAGE_MAP`
- 簡化三分支邏輯：`if in map → 使用共用函式`
- 刪除複雜的命令列印邏輯，改用 `_print_execute_command()`
- 函式行數：54 → 37（包含簡化後的文件字串）
