## 修改詳情

### 新增統一實作
- **檔案**: hook_utils.py
- **函式**: save_check_log(hook_name, log_content, logger)
- **簽名**: 通用設計，接受 hook_name 和日誌內容字串，自動建立日誌目錄和檔案

### 修改方式
- 移除每個 hook 中的本地 save_check_log 定義（30-40 行）
- 新增 save_check_log 至 import 清單
- 將本地呼叫改為統一呼叫，格式為：
  save_check_log("hook-name", log_entry, logger)

### DRY 原則達成
- 消除 8 個完全相同的函式實作
- 統一日誌格式和 hook-logs 目錄結構
- 易於維護和修改日誌策略
