#### 4. 邊界條件完整設計

**寬度計算邊界**（`calculate_visual_width`）：
- 空字串 (0)、單字元 (1 或 2)、多字元
- 純英文 (1 per char)、純中文 (2 per char)、混合
- 全形標點 (2)、emoji (2)、日韓字元 (2)

**填充邊界**（`ljust_with_chinese_width`）：
- 寬度不足：補充空格
- 寬度達到：無補充
- 寬度超過：保留原文本（不截斷）
- 目標寬度 0：無補充

**截斷邊界**（`truncate_title`）：
- max_length=0：返回空字串
- max_length < 2：無法容納 ".."
- 截斷位置在字元邊界
- 省略符寬度計入總寬度

**樹狀結構邊界**（`build_tree_structure`）：
- 無子任務：只有根任務清單
- 孤兒子任務：父任務缺失時作為根任務
- 多層嵌套：支援無限深度
- 排序：按字典序升序
