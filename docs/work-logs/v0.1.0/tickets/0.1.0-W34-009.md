### 修改內容

重構 `.claude/hooks/command-entrance-gate-hook.py` 中的 `SHORT_ANSWER_PATTERNS` 從 `list` 改為 `frozenset`：

1. **第 135-147 行**: 將常數定義從列表改為 frozenset
   - 原：`SHORT_ANSWER_PATTERNS = [...]`（28 個元素）
   - 新：`SHORT_ANSWER_PATTERNS = frozenset([...])`

2. **使用位置**: 第 321 行
   - 語法：`if prompt_stripped.lower() in SHORT_ANSWER_PATTERNS:`
   - 完全兼容 frozenset 的 IN 操作（無依賴列表順序的邏輯）
